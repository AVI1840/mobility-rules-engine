## תיאור מקצועי לסמנכ"לית — מנוע זכויות לניידות (POC)

**הבעיה**

תחום הניידות נשען על שכבות של מקורות מקצועיים: הסכם ניידות בן 58 עמודים, תדריך מקצועי, עשרות חוזרים, פסקי דין ותיקונים. הפקיד נדרש לאתר את המקור הרלוונטי, לפרש אותו ולהחיל אותו על המקרה — תהליך מורכב, עתיר זמן, ותלוי בניסיון אישי. התוצאה: פערי יישום בין סניפים, חוסר אחידות בהחלטות, וקושי בהכשרת פקידים חדשים.

הבעיה הזו קיימת בכל תחומי הזכויות בביטוח הלאומי — ניידות הוא ה-MVP.

**השיטה**

מנוע כללים דטרמיניסטי שמקודד את המקורות המקצועיים ומחליף את החיפוש הידני בלוגיקה מוסברת:

1. **קידוד** — כל חוזר, סעיף בהסכם ופסק דין מתורגם לכלל דטרמיניסטי עם ציטוט משפטי
2. **הרצה** — הפקיד מזין נתוני מקרה, המנוע מריץ את הכללים הרלוונטיים ומייצר המלצה מנומקת
3. **פידבק** — הפקיד מאשר, מתקן או מסמן חוסר דיוק — המערכת משתפרת בהדרגה
4. **אוטונומיה הדרגתית** — מקרים חד-משמעיים עוברים לטיפול אוטומטי, מקרים מורכבים נשארים בשיקול דעת

אין בינה מלאכותית גנרטיבית במסלול ההחלטה. כל קביעה מבוססת על סעיף משפטי מזוהה.

**מה כבר קיים**

| רכיב | סטטוס |
|-------|--------|
| הסכם הניידות — סעיפים 4, 5, 9א, 11, 13, 14, 20, תוספות ג'-ד' | ✅ מקודד |
| תדריך ניידות — חסר רכב, כפל גמלאות, הלוואות, תזמון החלפה | ✅ מקודד |
| מחשבון ניידות רשמי — עץ החלטות מלא עם קודי סכום | ✅ שוחזר כאשף אינטראקטיבי |
| 8 חוזרים (1810, 1905, 1931, 1936, 1984, 1996, 2056, 2132) | ✅ מקודדים |
| 4 פסקי דין (שושנה לוי, שקד ארועטי, גלית לביא, רות הדאיה) | ✅ מקודדים |
| הסכם קרן הלוואות + חוות דעת המשך תשלום | ✅ מקודדים |
| מנוע כללים דטרמיניסטי עם היררכיה משפטית | ✅ פעיל |
| ממשק מקצועי בעברית (RTL) | ✅ פעיל |
| מערכת פידבק מפקידים | ✅ פעילה |
| ולידציה מול מקרים היסטוריים | ✅ תשתית מוכנה |

**סה"כ: 32 כללים פעילים מ-17 מקורות מקצועיים**

**ערך אפשרי**

- **עקביות** — אותם נתונים מייצרים אותה המלצה, ללא תלות בפקיד
- **שקיפות** — כל החלטה מלווה בהנמקה וציטוט, ניתנת לביקורת
- **הכשרה** — פקיד חדש מקבל את כל הידע המקצועי מיום ראשון
- **בקרת איכות** — אפשרות להריץ בדיקות רוחב על החלטות קיימות
- **הרחבה** — המבנה מאפשר הוספת תחומים (סיעוד, נכות כללית, הכנסה)

**שלב הבא מוצע**

הצגה קצרה (15 דקות), ואם רלוונטי — בדיקת היתכנות מקצועית על מדגם מקרים אנונימיים.

**קישור:** https://avi1840.github.io/mobility-rules-engine/
